===========
Overview :-
===========

This is an online Mechanical Spur Gear Designer.

===============
Requirements :-
===============

1) LAMP server
2) Imagemagick
3) Inkscape
4) Zip
5) Python
6) pstoedit

========
Usage :-
========

1) Just copy all the files into your public_html or www folder.
2) Change folder permissions to 777
3) Access it through browser
